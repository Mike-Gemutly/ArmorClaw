# ArmorClaw User Guide

> User guide for ArmorClaw - Secure E2E Encrypted Chat Application

## 📖 Getting Started

### Installation

1. **Download ArmorClaw** from Google Play Store (or install APK)
2. **Open the app** and complete onboarding
3. **Create account** or **log in** with existing credentials
4. **Enable biometric auth** for secure unlock
5. **Start chatting!**

### First-Time Setup

#### Step 1: Welcome Screen
- See the features of ArmorClaw
- Tap **"Get Started"** to begin onboarding
- Tap **"Skip"** to go directly to login

#### Step 2: Security Explanation
- Learn about end-to-end encryption
- See animated security diagram (4 steps)
- Tap **"Next"** to proceed
- Tap **"Back"** to return

#### Step 3: Connect to Server
- Enter Matrix server URL (e.g., `matrix.org`)
- Tap **"Connect"** to connect to server
- Or tap **"Use Demo Server"** for demo mode

#### Step 4: Grant Permissions
- Grant **required permissions** (Network, Storage)
- Grant **optional permissions** (Camera, Microphone, Biometrics)
- See progress bar at top
- Tap **"Next"** when done

#### Step 5: Completion
- See celebration animation with confetti
- Read what's next (create room, join room, explore settings)
- Tap **"Start Chatting"** to go to home screen

## 🔐 Authentication

### Login

1. **Open ArmorClaw**
2. **Enter username or email**
3. **Enter password**
4. Tap **"Log In"** button
5. OR tap **"Log in with Biometrics"** for Fingerprint/FaceID

### Biometric Authentication

1. **Enable biometric auth** in Settings
2. **Set up fingerprint/FaceID** in device settings
3. **Use biometric** for quick app unlock
4. **Fallback to password** if biometric fails

### Forgot Password

1. **On Login screen**, tap **"Forgot password?"**
2. **Enter email** to reset password
3. **Check email** for reset link
4. **Reset password** and log in

### Registration

1. **On Login screen**, tap **"Don't have an account?"**
2. **Tap "Register"** to create new account
3. **Enter username, email, password**
4. **Verify email** (if required)
5. **Log in** with new credentials

## 🏠 Home Screen

### Room List

The home screen shows your chat rooms organized into categories:

- **Favorites** - Pinned favorite rooms
- **Chats** - All active chat rooms
- **Archived** - Archived rooms (collapsed)

### Room Cards

Each room card shows:
- **Room avatar** - Room avatar or initial
- **Room name** - Name of the room
- **Last message** - Preview of last message
- **Timestamp** - Relative time (2m, 1h, 1d, etc.)
- **Unread badge** - Unread message count
- **Encryption indicator** - Lock icon if encrypted

### Actions

- **Tap room** - Open chat
- **Tap search icon** - Search rooms
- **Tap profile icon** - Open profile
- **Tap settings icon** - Open settings
- **Tap FAB (+)** - Create new room
- **Tap "Join a room"** - Join existing room

## 💬 Chat Screen

### Sending Messages

1. **Type message** in input field at bottom
2. **Tap send button** (paper plane icon)
3. **Message appears** in chat list

### Message Status

Messages show status icons:
- **Sending** (clock icon) - Message is being sent
- **Sent** (check icon) - Message sent to server
- **Delivered** (double check) - Message delivered to recipients
- **Read** (blue double check) - Message read by recipients
- **Failed** (red exclamation) - Message failed to send

### Long Press Actions

**Long press on message** to see options:
- **Reply** - Reply to message
- **Forward** - Forward to other room
- **Add Reaction** - Add emoji reaction
- **Copy** - Copy message text
- **Delete** - Delete message (if sent by you)

### Replying to Messages

1. **Long press on message** you want to reply to
2. **Tap "Reply"**
3. **Type your reply** in input field
4. **Tap send button**
5. **Reply appears** with original message quoted

### Adding Reactions

1. **Long press on message** you want to react to
2. **Tap "Add Reaction"**
3. **Select emoji** from emoji picker
4. **Reaction appears** on message
5. **Tap reaction again** to remove

### Attachments

**Send files:**
1. **Tap attachment icon** (paper clip)
2. **Select file** from device
3. **Tap send button**

**Send images:**
1. **Tap attachment icon** (paper clip)
2. **Select image** from device or camera
3. **Add caption** (optional)
4. **Tap send button**

### Voice Messages

**Record voice message:**
1. **Tap microphone icon** and hold
2. **Speak** your message
3. **Release** to send

**Voice-to-text input:**
1. **Tap microphone icon** (once)
2. **Speak** your message
3. **Text appears** in input field
4. **Edit and send** if needed

### Searching Messages

1. **Tap search icon** in top bar
2. **Type search query**
3. **See search results** highlighted
4. **Tap result** to jump to message

### Encryption

ArmorClaw messages are **end-to-end encrypted**:
- **Lock icon** appears on encrypted messages
- **Encryption status** shows in top bar
- **All messages** are encrypted by default

## 🏠 Room Management

### Creating a Room

1. **Tap FAB (+)** in home screen
   OR **Tap "Join a room"** and select "Create Room" tab
2. **Tap "Create Room"** tab
3. **Enter room name** (required)
4. **Enter room topic** (optional)
5. **Set privacy** - Private or Public
6. **Add room avatar** (optional)
7. **Tap "Create Room"**

**Room Privacy:**
- **Private** - Only invited users can join
- **Public** - Anyone can join via room link

### Joining a Room

1. **Tap FAB (+)** in home screen
   OR **Tap "Join a room"** and select "Join Room" tab
2. **Tap "Join Room"** tab
3. **Enter Room ID** (required)
4. **Enter Room Alias** (optional)
5. **Tap "Join Room"**

**Finding Room ID:**
- Ask room owner for Room ID
- Room ID format: `!room:matrix.org`
- Room Alias format: `#room:matrix.org`

### Room Settings

**Access room settings:**
1. **Open chat** for the room
2. **Tap room name** in top bar
3. **See room settings:**
   - **Room name** - Edit room name
   - **Room topic** - Edit room topic
   - **Room avatar** - Edit room avatar
   - **Room privacy** - Change privacy
   - **Members** - View room members
   - **Leave room** - Leave the room
   - **Archive room** - Archive the room

## ❓ FAQ

### General

**Q: Is ArmorClaw free?**
A: Yes, ArmorClaw is completely free to use.

**Q: Is ArmorClaw open source?**
A: Yes, ArmorClaw is open source (MIT License).

**Q: Can I use ArmorClaw on iOS?**
A: iOS version is coming soon (KMP).

### Security

**Q: Are my messages encrypted?**
A: Yes, all messages are end-to-end encrypted (ECDH+AES-GCM).

**Q: Can ArmorClaw read my messages?**
A: No, messages are encrypted on your device. We can't read them.

**Q: Is biometric auth secure?**
A: Yes, biometric data never leaves your device.

### Features

**Q: Can I send files?**
A: Yes, you can send files, images, and voice messages.

**Q: Can I create private rooms?**
A: Yes, you can create private rooms where only invited users can join.

**Q: Can I archive rooms?**
A: Yes, you can archive rooms to hide them from main list.

### Troubleshooting

**Q: Messages not sending?**
A: Check internet connection, retry sending, or check server status.

**Q: Notifications not working?**
A: Check notification settings in app and device settings.

**Q: Biometric auth not working?**
A: Check device settings, ensure fingerprint/FaceID is set up.
