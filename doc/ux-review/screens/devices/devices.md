## User Flow

1. **User arrives from:** Settings screen
2. **User can:**
   - View all logged-in devices
   - See verification status
   - Verify unverified devices
   - Remove devices
   - Add new device
3. **User navigates to:**
   - Emoji verification screen
   - Add device screen
   - Settings screen (back)

## Notes

- Current device always shown first
- Unverified devices highlighted
- Verification crucial for E2E security
- Device removal revokes encryption keys
