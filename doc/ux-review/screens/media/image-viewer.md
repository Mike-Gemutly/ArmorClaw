## User Flow

1. **User arrives from:** Chat screen (tap image)
2. **User can:**
   - View full-size image
   - Zoom and pan
   - Download to device
   - Share to other apps
   - View image info
   - Dismiss viewer
3. **User navigates to:**
   - Chat screen (back/swipe)

## Accessibility

- **Content descriptions:**
  - Image: "Image from [sender]"
  - Download: "Download image"
  - Share: "Share image"
  - More: "More options"

- **Touch targets:**
  - Control buttons: 48.dp
