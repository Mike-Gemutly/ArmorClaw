## Best Practices

### For Users

1. **Set up a recovery key immediately** - Store it in a password manager or write it down
2. **Verify new devices promptly** - Unverified devices can't read encrypted messages
3. **Remove old devices** - Keep your device list clean and secure
4. **Transfer trust anchor before replacing** - Verify new device before removing old trust anchor

### For Administrators

1. **Require verification** - Configure homeserver to require device verification
2. **Monitor unverified devices** - Alert on accounts with many unverified logins
3. **Provide recovery options** - Ensure users know about recovery keys
4. **Document recovery process** - Make recovery procedures easy to find
