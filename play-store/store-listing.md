## Full Description (4000 characters max)

**Secure Communication for Modern Teams**

ArmorChat is an enterprise-grade encrypted messaging platform designed for organizations that demand the highest level of data security. Built with military-grade encryption, ArmorChat ensures your team's conversations remain completely private.

**Why Teams Choose ArmorChat:**

**UNBREAKABLE ENCRYPTION**
- End-to-end encryption using ECDH key exchange + AES-256-GCM
- Zero-knowledge architecture - we cannot read your messages
- SQLCipher encrypted database with 256-bit passphrase
- Secure key storage via AndroidKeyStore

**ENTERPRISE SECURITY FEATURES**
- Biometric authentication (Fingerprint/Face unlock)
- Secure clipboard with automatic clearing
- TLS with certificate pinning (SHA-256)
- Session timeout and auto-lock

**SEAMLESS TEAM COLLABORATION**
- Create and manage chat rooms
- Rich messaging: reactions, replies, attachments
- Voice messages for quick communication
- Message search across conversations
- Offline support - work without connectivity

**BUILT FOR BUSINESS**
- Clean, professional Material Design 3 interface
- Full offline capabilities with intelligent sync
- Accessibility compliant (TalkBack support)
- Dark mode for reduced eye strain

**PRIVACY FIRST**
- No data mining or advertising
- No third-party trackers
- Self-hosted server option
- Complete data ownership

**RELIABLE PERFORMANCE**
- Optimized for low battery and data usage
- Smooth animations and transitions
- Comprehensive crash reporting for stability
- 75+ automated tests ensuring quality
