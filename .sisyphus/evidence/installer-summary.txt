## Remaining Recommendations (Non-Blocking)

### 1. Add ngrok Transient Domain Warning (MEDIUM)
**Location:** setup-quick.sh after ngrok success
**Impact:** Users unaware mobile clients break on restart

**Suggested Addition:**
```bash
echo -e "${YELLOW}⚠ WARNING: ngrok domains change on restart${NC}"
echo -e "${YELLOW}  Mobile clients will need reconfiguration${NC}"
echo -e "${YELLOW}  For production, use Cloudflare or static domain${NC}"
```

---

### 2. Add Bridge Healthcheck to docker-compose-full.yml (LOW)
**Location:** docker-compose-full.yml bridge service

```yaml
healthcheck:
  test: ["CMD-SHELL", "test -S /run/armorclaw/bridge.sock || exit 1"]
  interval: 30s
  timeout: 5s
  retries: 3
```

---

### 3. Show Build Errors on Failure (LOW)
**Location:** setup-quick.sh build step

```bash
CGO_ENABLED=1 go build -o armorclaw-bridge ./cmd/bridge 2>"$WORK_DIR/build.log" &
if [[ ! -f "armorclaw-bridge" ]]; then
    print_error "Bridge build failed:"
    cat "$WORK_DIR/build.log" | tail -20
    fail
fi
```

---

### 4. Prefer IPv4 in IP Detection (LOW)
**Location:** Multiple files using `hostname -I | awk '{print $1}'`

```bash
ip_address=$(hostname -I | awk '{for(i=1;i<=NF;i++) if($i ~ /^[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+$/) {print $i; exit}}')
```

### 7. Consider Automatic Rollback on Failure (MEDIUM)
**Location:** All setup scripts

```bash
cleanup_on_error() {
    print_error "Installation failed, cleaning up..."
    systemctl stop armorclaw-bridge 2>/dev/null || true
    docker rm -f armorclaw-conduit armorclaw-tunnel 2>/dev/null || true
}
trap cleanup_on_error ERR
```
