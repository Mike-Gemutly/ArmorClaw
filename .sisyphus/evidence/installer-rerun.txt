## Identified Gap

### Gap: No Automatic Rollback
**Severity:** MEDIUM
**Impact:** Partial state may remain after failed installation

**Current Behavior:** Script exits, leaves partial state
**Recommendation:** Consider `trap 'cleanup_on_error' ERR` pattern:

```bash
cleanup_on_error() {
    print_error "Installation failed, cleaning up..."
    systemctl stop armorclaw-bridge 2>/dev/null || true
    docker rm -f armorclaw-conduit armorclaw-tunnel 2>/dev/null || true
    # Remove partial configs?
}
trap cleanup_on_error ERR
```

**Trade-off:** May remove user's existing valid configs on error
