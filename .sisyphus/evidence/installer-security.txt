## 1. GPG Signature Verification (Stage-0)

### install.sh (lines 102-131)

**Security Measures:**

1. **Signing Key Fingerprint Pinning** (line 16):
   ```bash
   SIGNING_KEY_FPR="55AD64228EF6B4A342DA480A09C43CFA8AC93062"
   ```
   - Hardcoded fingerprint prevents key replacement attacks
   - Key must match expected fingerprint exactly

2. **Temporary GPG Home** (lines 104-106):
   ```bash
   mkdir -p "$GNUPGHOME"
   chmod 700 "$GNUPGHOME"
   ```
   - Isolated keyring in temp directory
   - 700 permissions prevent other users from accessing

3. **Fingerprint Verification** (lines 116-122):
   ```bash
   FPR_CHECK=$(gpg --homedir "$GNUPGHOME" --with-colons --fingerprint releases@armorclaw.ai | grep "^fpr" | head -n1 | cut -d: -f10)
   if [[ "${FPR_CHECK:-}" != "$SIGNING_KEY_FPR" ]]; then
       echo -e "${RED}ERROR: Unauthorized signing key detected!${NC}"
       exit 1
   fi
   ```
   - Verifies fingerprint BEFORE trusting the key
   - Prevents loading malicious keys

4. **Signature Verification** (lines 125-130):
   ```bash
   if ! gpg --homedir "$GNUPGHOME" --batch --verify "$SIG_PATH" "$INSTALL_PATH"; then
       echo -e "${RED}ERROR: GPG signature verification failed!${NC}"
       exit 1
   fi
   ```
   - Batch mode prevents interactive prompts
   - Explicit error message on failure

**Evidence Quality: EXCELLENT**
- Defense-in-depth: SHA256 + GPG + fingerprint
- No trust-on-first-use for keys
