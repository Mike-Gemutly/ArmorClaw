## 3. Download Failure Handling

### install.sh - Retry Logic (lines 27-40)
```bash
retry() {
    local n=1
    local max=3
    local delay=2
    while true; do
        "$@" && return 0
        if (( n == max )); then
            return 1
        fi
        echo -e "  [armorclaw] Retrying in ${delay}s... ($n/$max)"
        sleep $delay
        ((n++)) || true
    done
}
```

**Features:**
- 3 retry attempts
- 2 second delay between retries
- Progress indication (n/max)
- Returns failure code after max retries

**Evidence Quality: EXCELLENT**
- Resilient to transient network issues
- User sees progress

### Gap 1: No Rollback on Partial Failure
**Severity:** MEDIUM
**Impact:** If installation fails mid-way, partial state may remain

**Current Behavior:** Script exits, leaving installed components
**Recommendation:** Consider `trap 'cleanup_on_error' ERR` to remove partial state
