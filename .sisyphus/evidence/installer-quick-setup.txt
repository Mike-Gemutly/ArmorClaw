### Step 2: Bridge Installation (lines 805-885)

**Options:**
1. **Binary download** (preferred) - Downloads prebuilt binary from GitHub releases
2. **Source build** (fallback) - Clones repo and builds with CGO/SQLCipher

**Binary Download Flow:**
- Detect architecture (linux-amd64, linux-arm64)
- Download binary + checksums.txt + checksums.txt.sig
- Verify SHA256 checksum
- Verify GPG signature with fingerprint pinning
- Install to /opt/armorclaw/armorclaw-bridge

**Source Build Flow:**
- Install Go 1.22+ if needed
- Clone repo with --depth 1
- Build with `CGO_ENABLED=1` for SQLCipher
- Install to /opt/armorclaw/

**Existing Installation Handling:**
```bash
if [[ -x "$INSTALL_DIR/armorclaw-bridge" ]]; then
    print_done "Bridge already installed"
    if ! $NON_INTERACTIVE && prompt_yes_no "Reinstall bridge?" "n"; then
        $SUDO rm -f "$INSTALL_DIR/armorclaw-bridge"
    else
        return 0  # Skip reinstall
    fi
fi
```

**Evidence:** PASS - Safe reinstall handling

## Identified Issues

### Issue 1: IPv6 May Be Returned Before IPv4 (LOW)
**Location:** Line 369, 388, 1274
**Code:** `hostname -I | awk '{print $1}'`
**Impact:** May return IPv6 address which some clients don't handle
**Fix:**
```bash
hostname -I | awk '{for(i=1;i<=NF;i++) if($i ~ /^[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+$/) {print $i; exit}}'
```
